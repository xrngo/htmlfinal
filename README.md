# Web Application

## Структура проекта
```text
webapp/
├── backend/ # Backend API сервер
│ ├── server.js # Основной серверный файл
│ ├── database.js # Работа с базой данных
│ ├── package.json # Зависимости backend
│ ├── README.md # Документация backend
├── .gitignore # Git ignore
├── README.md # Общая документация
└── test_api.html # HTML документ для тестирования работы с API
```

## Установка

* Клонируйте репозиторий
```bash
git clone https://gitverse.ru/Belisarius/Web-Application.git
```

* Для дальнеших действий воспользуйтесь инструкцией расположенной в README.md директории backend
